## 功能特性

 - 第三方OAuth配置使用
 - 依托Web UI的用户管理系统
 - 跨平台多终端集成能力（即将支持Android、iOS）
 - 基于HTTPS和JWT的安全认证方式
 - 自定义邮件模版、自定义第三方邮件服务
 - Docker微服务架构，拥有99.9%的服务可用性
 - 基于GraphQL的消息通信
