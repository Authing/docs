# Remove User

----------

#### Authing.remove(uid)

- **Arguments:**

  - ```{String} uid```

- **Usage:**

  - ``` javascript
  let uid = '59e5ff4935eebf1913cfe8a1';
	Authing.remove(uid);
  	```
- **returns:**

  - ``` javascript

	{
		code: 1, //-1 represents failed
		data: null,
		message: 'remove success'
	}

    ```
