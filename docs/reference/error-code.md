# 错误代码

错误代码消息格式如下：

```js
{
  "uniqueId": "65e59f51-dbbc-4d1f-a809-6d3f15d9b53e", // 唯一标识，用于定位错误
  "statusCode": 499, // 状态码
  "message": "应用不存在", // 错误信息
  "apiCode": "2011", // 和业务绑定
  "data": null // 附加数据
}
```
