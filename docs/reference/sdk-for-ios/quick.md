## 第二步：添加 Guard 依赖

在 Xcode 菜单栏右上角选择 *File* 点击 *Add Packages*

在 Swift Package 搜索栏输入：https://github.com/Authing/guard-ios

依赖规则选择 *Up to Next Major Version 1.0.0*
