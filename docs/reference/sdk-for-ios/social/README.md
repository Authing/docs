# Social API

<LastUpdated/>

在使用 Authing 之前，每个 App 都需要对接每个社会化登录的平台和接口。服务端需要对接平台的后台接口，移动端需要集成各平台的 SDK。

通过 Guard SDK 开发者只需要和 Authing 对接一次，就可以轻松配置多个第三方身份源。

我们提供了以下第三方身份源的便捷登录授权：

- [飞书登录](./lark.md)
