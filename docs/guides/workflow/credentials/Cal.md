 使用API密钥
 [#](#使用api键 "永久链接")
-------------------------------------------------------



 您的API请求使用API密钥进行身份验证。任何不包含API密钥的请求都将返回错误。
 


1. 登录您的Cal帐户。
2. 单击
 **设置**
 .
3. 在那里，您可以找到API密钥部分，该部分允许您生成密钥以供使用。有关这方面的更多信息，请访问
 [Cal认证](https://developer.cal.com/api/authentication) 
 页
4. 在n8n中的Cal触发器节点凭据中使用API密钥。
