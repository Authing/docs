 先决条件
 [#](#先决条件 "永久链接")
-----------------------------------------------------



 创建

 账户
 



 使用访问令牌
 [#](#使用访问令牌 "永久链接")
---------------------------------------------------------------


1. 打开Coda仪表板。
2. 单击右上角的个人资料。
3. 单击3个点，然后单击帐户设置。。
4. 向下滚动并单击API下的“生成”。
5. 在n8n中的Coda节点凭据中使用选定的API访问令牌。
