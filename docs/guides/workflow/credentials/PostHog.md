=========================================



 您可以使用这些凭据通过PostHog验证以下节点。
 


*[PostHog]（/integrations/builtin/app nodes/n8n nodes-base.postog/）



 先决条件
 [#](#先决条件 "永久链接")
-----------------------------------------------------



 创建
 [邮政编码](https://posthog.com/) 
 帐户或主机PostHog。
 



 使用API密钥
 [#](#使用api键 "永久链接")
-----------------------------------------------------


1. 打开PostHog仪表板。
2. 单击
 ***项目***
 在左侧边栏上。
3. 单击
 ***安全与隐私***
 选项卡。
4. 向下滚动到
 ***项目API密钥***
 部分，并复制显示的API密钥。
5. 在
 ***凭据名称***
 n8n中“PostHog API”凭据中的字段。
6. 如果您在自己的服务器上托管PostHog，请在
 ***网址***
 字段。
7. 将API密钥粘贴到
 ***API密钥***
 n8n中“PostHog API”凭据中的字段。
8. 单击
 ***创建***
 按钮创建凭据。
