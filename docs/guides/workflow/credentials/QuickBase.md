 使用API
 [#](#使用api "永久链接")
---------------------------------------------


1. 访问Quick Base仪表板。
2. 单击右上角的姓名，然后从下拉列表中选择“我的首选项”。
3. 单击
 ***管理我的用户令牌***
 在
 ***我的用户信息***
 部分
4. 单击
 ***+新用户令牌***
 按钮
5. 在
 ***姓名***
 领域
6. 从
 ***将令牌分配给应用***
 下拉列表。
7. 单击
 ***保存***
 按钮
8. 复制位于
 `https://`
 和
 `/数据库`
 在Quick Base URL中。此字符串是主机名。
9. 使用此
 ***主机名***
 和
 ***用户令牌***
 使用n8n中的Quick Base节点凭据。
