 使用API密钥
 [#](#使用api键 "永久链接")
-----------------------------------------------------


1. 打开DeepL
 [帐户](https://www.deepl.com/pro-account) 
 页
2. 向下滚动到
 ***DeepL API的身份验证密钥***
 部分
3. 复制显示的“API密钥”。
4. 在
 ***凭据名称***
 n8n中“DeepL API”凭据中的字段。
5. 将API密钥粘贴到
 ***API密钥***
 n8n中“DeepL API”凭据中的字段。
6. 单击
 ***创建***
 按钮在n8n中创建凭据。
