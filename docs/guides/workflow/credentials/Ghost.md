


 Ghost
 [#](#ghost "Permanent link")
=====================================



 You can use these credentials to authenticate the following nodes with Ghost.
 


* [Ghost](/integrations/builtin/app-nodes/n8n-nodes-base.ghost/)



 Prerequisites
 [#](#prerequisites "Permanent link")
-----------------------------------------------------



 Create a
 [Ghost](https://ghost.org/) 
 account.
 



 Using Admin API
 [#](#using-admin-api "Permanent link")
---------------------------------------------------------


1. Access your Ghost Admin dashboard.
2. Click on
 ***Integrations***
 in the left sidebar.
3. Click on
 ***+ Add custom integration***
 under the
 ***CUSTOM INTEGRATIONS***
 section.
4. Enter a name for the integration in the
 ***Name***
 field.
5. Click on the
 ***Create***
 button.
6. Use this
 ***URL***
 and
 ***Admin API Key***
 with your Ghost node credentials in n8n.



 Using Content API
 [#](#using-content-api "Permanent link")
-------------------------------------------------------------


1. Access your Ghost Admin dashboard.
2. Click on
 ***Integrations***
 in the left sidebar.
3. Click on
 ***+ Add custom integration***
 under the
 ***CUSTOM INTEGRATIONS***
 section.
4. Enter a name for the integration in the
 ***Name***
 field.
5. Click on the
 ***Create***
 button.
6. Use this
 ***URL***
 and
 ***Content API Key***
 with your Ghost node credentials in n8n.
