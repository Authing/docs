 [#](#使用api键 "永久链接")
-----------------------------------------------------



 成为行动网络合作伙伴后：
 


1. 打开行动网络仪表板。
2. 从
 **开始组织**

 **详细信息**
 >
 **API和同步**
 .
3. 使用下拉列表选择要为其生成密钥的列表、您的个人列表或您是其管理员的任何组的列表。
4. 从
 **您的API密钥**
 部分，生成新的API密钥：
