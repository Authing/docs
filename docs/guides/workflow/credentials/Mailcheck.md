


 Mailcheck
 [#](#mailcheck "Permanent link")
=============================================



 You can use these credentials to authenticate the following nodes with Mailcheck.
 


* [Mailcheck](/integrations/builtin/app-nodes/n8n-nodes-base.mailcheck/)



 Prerequisites
 [#](#prerequisites "Permanent link")
-----------------------------------------------------



 Create a
 [Mailcheck](https://app.mailcheck.co/) 
 account.
 



 Using API Key
 [#](#using-api-key "Permanent link")
-----------------------------------------------------


1. Open the Mailcheck
 [dashboard](https://app.mailcheck.co/dashboard) 
 page.
2. Click on the API on the top menu.
3. Enter a key name in the
 ***Enter the key name***
 field.
4. Click on the
 ***Create Key***
 button.
5. Click on the
 ***Copy***
 button to copy the API key.
6. Enter a name for your credentials in the
 ***Credentials Name***
 field in the 'Mailcheck API' credentials in n8n.
7. Paste the API key in the
 ***API Key***
 field in the 'Mailcheck API' credentials in n8n.
8. Click on the
 ***Create***
 button to create your credentials in n8n.
