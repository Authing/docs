 您可以使用这些凭据通过Dropcontact验证以下节点。
 


*[Dropcontact]（/integrations/builtin/app nodes/n8n nodes base.Dropcontact/）



 先决条件
 [#](#先决条件 "永久链接")
-----------------------------------------------------



 在上创建帐户
 [下拉联系人](https://www.dropcontact.com/) 
 应用程序。
 



 使用API密钥
 [#](#使用api键 "永久链接")
-----------------------------------------------------


1. 打开Dropcontact
 [仪表板](https://app.dropcontact.io/app/) 
 .
2. 选择
 ***API接口***
 从左侧边栏。
3. 单击
 ***复制***
 按钮复制API密钥。
4. 将API密钥粘贴到
 ***API密钥***
 n8n中“Dropcontact API凭据”字段中的字段。
5. 单击
 ***保存***
 按钮创建凭据。
