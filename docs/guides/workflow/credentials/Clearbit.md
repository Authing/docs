


 Clearbit（清除位）
 [#](#clearbit "永久链接")
===========================================



 您可以使用这些凭据使用Clearbit对以下节点进行身份验证。
 


*[Clearbit]（/integrations/builtin/app-nodes/n8n-nodes-base.Clearbit/）



 先决条件
 [#](#先决条件 "永久链接")
-----------------------------------------------------



 创建
 [清除位](https://www.clearbit.com/) 
 账户
 



 使用访问令牌
 [#](#使用访问令牌 "永久链接")
---------------------------------------------------------------


1. 打开Clearbit仪表板。
2. 单击“API”。
3. 单击“Configure Chargebee”（配置充电器）。
4. 在n8n中的Clearbit节点凭据中使用选定的API密钥。
