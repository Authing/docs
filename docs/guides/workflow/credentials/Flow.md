 使用访问令牌
 [#](#使用访问令牌 "永久链接")
---------------------------------------------------------------


1. 访问流量仪表板。
2. 单击左上角的组织图标。
3. 单击帐户首选项。
4. 单击
 *创建个人访问令牌*
 按钮
5. 当您生成访问令牌时，您还将收到组织ID。
6. 将提供的访问令牌和组织ID与n8n中的Flow节点凭据一起使用。
