


 日期和时间
 [#](#日期时间 "永久链接")
===============================================



 “日期和时间”节点用于处理日期和时间数据，并将其转换为不同的格式。
 




 牢记
 



 确保为n8n实例（或工作流）正确设置了时区。
 




 节点参考
 [#](#节点引用 "永久链接")
-------------------------------------------------------


*行动
	+将日期转换为其他格式



 可以从
 *设置格式*
 下拉列表。
 


*要设置格式
	+年/月/日
	+年/月/日
	+年月日
	+年月日
	+年-月-日
	+Unix（x）时间戳
	+Unix Ms时间戳



 也可以通过设置
 *自定义格式*
 切换到“打开”。
 


*选项
	+*来自格式*
	 字段：允许您指定输入值的格式。请参阅
	 [常见问题解答]（#what-values-can-i-use-in-the-from-mat-field）
	 了解可能的吹嘘。
	+*来自时区*
	 字段：允许您为不同于n8n系统时钟的输入时区指定输入值的时区。
	+*至时区*
	 字段：允许您指定输入值必须转换到的时区。
