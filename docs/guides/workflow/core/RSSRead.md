


 RSS读取
 [#](#rss读取 "永久链接")
===========================================



 RSS读取节点用于从互联网上发布的RSS提要中读取数据。
 



 节点参考
 [#](#节点引用 "永久链接")
-------------------------------------------------------



 RSS读取节点只有一个属性：
 


**网址*
 field：此字段用于指定RSS出版物的网址。

 [#](#示例用法 "永久链接")
