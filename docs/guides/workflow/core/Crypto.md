


 加密
 [#](#crypto "永久链接")
=======================================



 加密节点用于加密工作流中的数据。
 



 节点参考
 [#](#节点引用 "永久链接")
-------------------------------------------------------



 在
 *类型*
 和
 *编码*
 字段。
 


*行动
	+以指定格式哈希文本
	+Hmac指定格式的文本
	+使用私钥对字符串进行签名
*类型
	+MD5型
	+沙256
		-沙3-256
		-沙384
		-沙3-384
	+沙512
		-沙3-512
*编码
	+基础64
	+十六进制
