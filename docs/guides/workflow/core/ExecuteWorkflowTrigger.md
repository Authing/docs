


 执行工作流触发器
 [#](#执行工作流触发器 "永久链接")
===========================================================================



 使用此节点可以启动工作流以响应其他工作流。它应该是工作流中的第一个节点。
 



 n8n允许您从其他工作流调用工作流。如果您想：
 


*重用工作流：例如，您可以有多个工作流从不同的源提取和处理数据，然后让所有这些工作流调用生成报告的单个工作流。
*将大型工作流拆分为较小的组件。



 此节点响应来自
 [执行工作流]（/integrations/builtin/core nodes/n8n nodes base.executeworkflow/）
 节点。
