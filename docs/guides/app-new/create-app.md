# 自建应用的创建

<LastUpdated/>

路径：**应用->自建应用**

您可以在 **自建应用** 页面查看所有自建应用。

<img src="./images/create-app-overview.png" alt="drawing"/>

要创建新的自建应用，执行以下步骤：

1. 点击 创建自建应用 按钮。

2. 点击页面右上角 **创建自建应用** 按钮。

3. 在弹出窗口填写如下信息：
    - **应用名称**: 应用名
    - **认证地址**: 输入二级域名，必须为合法的域名格式（只允许包含英文、数字和 -，例如 **my-awesome-app**
    - **选择类型**: 应用类型

4. 点击 **创建** 按钮。
