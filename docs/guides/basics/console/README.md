# 控制台概览

<LastUpdated/>

控制台是你管理和配置所有 Authing 资源的地方，这篇文章会帮助你学会使用 Authing 控制台提升生产力！

在 [Authing 控制台](https://console.authing.cn/console/userpool)中你可以对 Authing 的资源、用户等信息进行配置和修改。

![](./images/overview.png)

::: img-description
控制台概览
:::
