# SocialLoginListView

<LastUpdated/>

此组件根据控制台设置显示社会化登录源。用户点击图标时，拉起相应 App 完成认证。

![](./../images/social1.png)

## 属性

| 属性名                     | 类型 | 说明 | 默认值 |
| ----------------------- |:--------:| :------:| :-----: |
|  src     |    String    |   社会化登录源。auto：跟随控制台；wechat\|apple，用 \| 隔开   |    auto   |

## 设置属性示例

```swift
if let social: SocialLoginListView = Util.findView(view, viewClass: SocialLoginListView.self) {
    social.src = "wechat"
}
```
