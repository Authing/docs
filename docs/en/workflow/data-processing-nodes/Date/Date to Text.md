# Date to Text

Converts the date and time to the format provided in the specified time zone.

## Enter parameters

- `start` : date and time values to format.
- `format` : Date format code for the input value of the starting field (such as M/D/Y) to produce 9/7/2016 or hh:mm a, for example 09:05am. See the format code below.
- `zone` : TZ time zone, such as USA/Pacific or Europe/Paris.

## Output parameters

- `output` : Specifies the date text in the selected format in the time zone.
