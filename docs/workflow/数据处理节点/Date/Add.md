# Add

从起始日期和时间中添加时间单位(例如秒)的倍数，并返回一个新的日期值。

## 输入参数

- `start date`：开始日期和时间。
- `add`：要添加到开始日期和时间的金额。
- `units`：要添加到开始日期的特定单元类型(按比例计算)。选项有:

  - milliseconds（毫秒）
  - seconds（秒）
  - minutes (分钟)
  - Hours (小时)
  - Days (天)
  - weeks (周)
  - Months (月)
  - Years (年)

## 输出参数

- `output`：指定相加后的结果日期和时间。
