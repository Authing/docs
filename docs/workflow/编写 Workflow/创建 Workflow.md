# 创建 Workflow

创建一个 workflow 有两种方式：

1. 从零开始创建 workflow
2. 从 Authing 模版市场中选择 workflow 模版

![](../static/boxcnIHz4TnidO3UR4GI8ULJ4Eg.png)

输入 workflow 的名称和描述即可：

在 Authing 身份自动化的模版市场中，选择你想要创建的模版：
